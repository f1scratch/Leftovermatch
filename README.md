# Leftovermatch
ISU project

Have you ever wondered about what to cook for dinner? This is a common daily challenge, which I aim to solve with my app, LeftoverMatch. This app allows you to search with the ingredients you have (even if they're random), pick your dietary preference to suggest appropriate recipes, and enter your allergies to filter out any recipe that contains them. This recommendation engine makes your daily life easier by thinking for you, bringing you the recipes that fit your needs. This app is highly user-friendly and works effectively, with all of its core functionalities working.
